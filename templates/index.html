API Escuelas de Informática

Rutas

/

Retorna la pagina principal.

Method: GET
/getlugares/<usuario_nombre>
/getlugares

Retorna los lugares agregados por un usuario en particular o todos si usuario_nombre no es especificado.

Method: GET
/getlugar/<nombre_lugar>

Retorna un lugar cuyo nombre sea nombre_lugar.

Method: GET
/guardarlugar

Permite guardar un lugar en la base de datos. Los datos serán considerados válidos si tienen la siguiente forma:

{"nombre": "CASA", "latlng": "2222/222", "descripcion": "es mi lugar", "usuario":"pablo1n7"} Method: POST
